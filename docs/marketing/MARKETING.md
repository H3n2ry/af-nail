## Af.nail

---

## 1. Elevator Pitch

### Versão Técnica (devs/investidores)
> "Af.nail é um marketplace PWA React + Cloudflare Workers para agendamento de manicure com dois portais separados (cliente e profissional), modelo de assinatura mensal e cálculo automático de slots de disponibilidade."

### Versão Usuário Final
> "Quer marcar manicure sem ligar? Entra no Af.nail, busca o salão, escolhe o horário disponível, confirma. Recebe lembrete automático. Pronto."

### Versão Profissional
> "Cansada de gerenciar agenda por WhatsApp? No Af.nail sua cliente agenda sozinha nos horários que você já definiu. Você só aparece para atender."

---

## 2. Proposta de Valor

| | |
|-|-|
| **Problema** | Agendamento por WhatsApp é caótico — mensagens perdidas, conflito de horários, sem histórico |
| **Solução** | Agenda digital com disponibilidade configurável, booking online 24h e lembretes automáticos |
| **Diferencial** | Simples o suficiente para manicures autônomas; PWA sem download; preço acessível |
